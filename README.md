# 👟 Nike Store Website

**Live: https://nike-store-clone-sooty.vercel.app/**

***A website made to practice React, CSS skills by trying to replicate a Nike Store Website. Made it responsive to screens
of different sizes using **media queries**.*

## ⚒️ Built with:
- HTML 5
- CSS 3
- JavaScript
- React
- Vite
